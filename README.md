# Student Grade Sorter (x86 Assembly)

A console-based **Student Grade Management System** developed in **x86 Assembly Language** as part of the **Computer Organization & Assembly Language (COAL)** course.

The project demonstrates low-level programming concepts including memory management, register manipulation, modular procedure design, sorting algorithms, and input validation.

## 📌 Project Overview

The program allows users to enter student names and scores, sorts them in ascending or descending order using Bubble Sort, assigns letter grades, and generates a complete statistical report.

---

## ✨ Features

- Live student data entry
- Input validation
- Bubble Sort (Ascending & Descending)
- Automatic letter grade assignment
- Student ranking
- Highest score detection
- Lowest score detection
- Average score calculation
- Grade distribution report
- Error handling and re-prompts
